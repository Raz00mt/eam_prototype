# eam_prototype
Для запуска необходимо 
1. поднять PostgreSQL БД
docker compose up -d

2. запустить веб-приложение 
python -m streamlit run app.py 

Для обучения модели после запуска веб-приложения в терминале прописать команду
python -m src.ml.train
и после этого запустить приложение командой из пункта 2 или выполнить rerun через веб-интерфейс
